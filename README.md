# concurrency-tasks
Python concurrency tasks

## Відповіді

### Завдання 1
Race condition — це ситуація, коли кілька потоків змінюють одну змінну одночасно, через що результат неправильний.

### Завдання 2
Lock працює, бо дозволяє тільки одному потоку змінювати змінну.

Мінуси:
- повільніше
- можливі блокування (deadlock)

### Завдання 7
Найшвидший варіант — ProcessPool.

Причина:
Python має GIL (Global Interpreter Lock), тому потоки не можуть виконувати CPU-задачі паралельно.
Процеси працюють незалежно і використовують кілька ядер процесора.
